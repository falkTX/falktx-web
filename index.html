My Projects

Created Open-Source Projects

My own little open-source projects, mostly created from scratch.
Click on the images to see them in their original size and on the project title to open its page in a separate tab.

KXStudio

Linux distribution and repositories, focused on audio production.
Can be installed as full OS, or extend Debian/Ubuntu based systems through its repositories.

Cadence

A collection of tools to manage Linux audio.
Includes Audio settings, patchbay, recorder, meter and logs viewer.

Carla

Fully-featured cross-platform audio plugin host.
Loads plugins (LADSPA, DSSI, LV2, VST, AU) and sample banks (GIG, SF2, SFZ), and also works as an audio plugin itself.

DPF / DPF-Plugins

C++ framework to create real-time cross-platform audio plugins.
Provides UI support using OpenGL, with integrated NanoVG library.

DISTRHO-Ports

Linux ports of existing open-source audio plugins, using Juce.
Contains a forked version of the Juce library for LV2 plugin support.
